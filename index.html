Theater Finder

Ever wanted to get all information regarding a movie in one place?

Then this application is for you. Check out Theater Finder. It's a nice alternative to multiple Google Searches and clicks across random pages.



Installation

Open Terminal and run the following commands (if you already have git command line tools installed; if not, see Install Git): 

>>> git clone https://github.com/nikhilajjarapu/theater-finder.git
>>> cd theater-finder
>>> python main.py




Yes, that's it. Did you really expect more complicated instructions?
